Secure File Sharing

Cloud-Based Secure File Sharing System

Security Features:

  • Hybrid Cryptography: Combines RSA and AES for optimal security
  • End-to-End Encryption: Files are encrypted before transmission
  • Secure Key Exchange: Using Diffie-Hellman key exchange protocol
  • Database Security: PostgreSQL with encrypted storage

How It Works:

  1. Files are encrypted using AES (Advanced Encryption Standard)
  2. AES key is encrypted using recipient's RSA public key
  3. Encrypted data is securely stored in the cloud
  4. Only the intended recipient can decrypt using their private key
